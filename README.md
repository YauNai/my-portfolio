# 作品集網站

此網頁放置了我的履歷，以及我開始工作以來的所有作品，包含 3D 動畫和前端網頁程式設計作品。

[線上預覽](https://my-portfolio-gwg7.onrender.com)

## 網站特色

- 左右雙區塊：履歷區｜作品集
  - 使用者可自由切換各區塊顯示的畫面 
- 響應式設計
- 可互動的 3D 畫面
  
## 使用框架與套件
### 框架

- Vue.js

### 套件

- Vue Router
- Three.js
- GSAP
- TypeIt
- AOS

### 設計規劃
